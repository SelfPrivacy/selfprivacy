### How to fix a domain for Porkbun DNS
1. Visit the following [link](https://porkbun.com) and sign into your Porkbun account.
2. Open the "Domain Management" page from the account menu bar on the right.
3. For the domain you selected in SelfPrivacy, expand the details section by clicking on the arrow down button on the right side of the domain.
4. Make sure the domain is using Porkbun nameservers in the "Nameservers" section. Porkbun nameservers end with "porkbun.com".
5. In the same domain settings, enable API access for this domain.
5. Wait a moment for Porkbun to apply changes, then return to SelfPrivacy and try again.
