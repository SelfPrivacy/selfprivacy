### How to get Porkbun API credentials
1. Visit the following [link](https://porkbun.com) and sign into your Porkbun account.
2. Open **Account** -> **API Access**.
3. Generate a new API key pair if you do not have one yet.
4. Copy and save both values:
   - **API Key** (this is the Token ID field in SelfPrivacy)
   - **Secret API Key** (this is the Token field in SelfPrivacy)
5. On the Domain Management screen, make sure API access is enabled for the domain you want to use with SelfPrivacy. By default it is disabled, you have to explicitly enable it the domain you want to use with SelfPrivacy. You can find the API access toggle in the domain details, in the "API Access" section.
6. Return to SelfPrivacy and paste both credentials.
